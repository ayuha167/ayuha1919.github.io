Twitch API とサービス

Mar 27th, 2016

Twitch API は公開されています。デベロッパの方向け…程には書くスキルがないのですが、紹介いたします。 https://github.com/justintv/Twitch-API

Twitch API

もしかして、Twitchのことを書くのは初めてでしょうか？
というわけで、上のリンクでTwitchのAPIは出ています。これを用いて開発が可能です。

Twitch Alerts や Night Dev といった、生放送の体験を拡張する機能から、#TwitchPlaysPokemon で一度Twitterを沸かせた企画まで、色々出来ます。

最近リリースされたサービス

そんな中、最近のちょうさんがリリースしたスマキュレーションに、配信チェック機能が追加されました。
内容は以下のツイートをご参考に：

スマキュレーションにスマブラーのTwitch配信の一括視聴機能を追加しました。
「1クリックでチャンネル切替」「マルチウインドウ機能（多窓）」など、スマブラの配信が手軽に見られます。https://t.co/JwPqMIh4eY pic.twitter.com/NwlIJldQAt

— のちょう (@noticoco) March 23, 2016

機能１「チャンネル切替」

事前に登録してあるスマブラーの配信状態をチェックでき、１クリックで配信が切り替わります。
配信状態は一定間隔で自動更新されるので、リロードの必要はありません。
一部海外の有名プレイヤーも追加しています。 pic.twitter.com/mssmlkQu8X

— のちょう (@noticoco) March 23, 2016

機能２「マルチウインドウ」

右上の「マルチウインドウに切替」を押すことで、リストに載っているプレイヤーを同時視聴できます。
最大３窓まで設定を行えます。リストからドラッグ＆ドロップするだけです。
※ＰＣ版のみの機能です pic.twitter.com/dthxGwctSk

— のちょう (@noticoco) March 23, 2016

機能３　「スマートフォン視聴」

通常はTwitchはアプリをインストールしないと視聴できないのですが、ブラウザからそのまま視聴が行えるようにしました。
ただ、一部のiOSでしか確認が取れていないため、観れない場合もあるかも？ pic.twitter.com/QtUNnT7pQE

— のちょう (@noticoco) March 23, 2016

スマブラを主に配信される方は追加致します！
ニコ生派の方はTwitchも同時に配信すればいいんじゃないかな！
とゆうことで、みんなTwitchに登録してね！！！！！（熱いステマ）

— のちょう (@noticoco) March 23, 2016

Twitch は固定URLで永遠に配信が出来るため、HTMLでのウェブサイト埋め込みをはじめ、こんなこと↑まで可能にしています。
