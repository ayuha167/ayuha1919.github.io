Dec 23rd, 2013

ブログを遂に始めました。誤解があったかもしれない。「再開」しました。

GitHub と Octopress を連携させています。詳しくは以下で↓
Salingerさんのページ 酒と涙とRubyとRails

Salingerさんの.pdfが一番役に立ちました。あと下の酒と涙、というものも凄く納得いきました。 mac向けにも書いてあり、ruby のインストール等々で苦戦しましたが、しっかりこの通り動きました。

まだこれはテストページですが、今後ともよろしくお願いします。

環境

PC: MacBook Pro, Core i7, Mem 8GB, not Retina
OS: OS X 10.7.3
XCode: 4.6.2

ちなみに、備忘録

    $ rake new_post\[title\]

で記事が作成され、 source/_post/2013-XX-XX-1.markdown というディレクトリに記事のテンプレートが現れる。これを編集してページ作るのです。

Markdown記法に関しては、
kojika17さんのページ で。

簡単に言うと、

  • スペース2個で改行
  • HTML の <えいち1> が “シャープ”, <えいち2> が “シャープシャープ”

です。

あとは

    $ rake generate

で。
プレビュー見るなら

    $ rake preview

で、http://localhost:4000/ にアクセスすれば見られる。

ここでOKなら、

    $ rake deploy

で公開。
